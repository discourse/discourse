# Writing Discourse CSS

Use this skill for CSS/SCSS changes in Discourse core, bundled plugins, themes, and theme components.

Before writing any CSS/SCSS code, read the [Discourse CSS and BEM guidelines](https://github.com/discourse/discourse/blob/main/docs/developer-guides/docs/03-code-internals/25-css-guidelines-bem.md) and apply those conventions.
