# mutant skill

`SKILL.md` is our own short guide to acting on mutant's output. The methodology
follows [mutant's own docs](https://github.com/mbj/mutant) (add a test, adopt the
simpler code, or accept an equivalent), written in our words so nothing is copied
in. Repo-specific setup — how to run it, what is in scope, gotchas — lives in
[`references/migrations-tooling.md`](references/migrations-tooling.md).
