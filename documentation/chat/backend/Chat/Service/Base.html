.contract(&block) ⇒ Object

Checks the validity of the given context. Supports after/before/around callbacks. Implements ActiveModel::Validations and ActiveModel::Attributes.

Examples:

before_contract {}
around_contract {}
after_contract {}

contract do
  attribute :name
  validates :name, presence: true
end
